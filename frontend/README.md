# 多智能体协作系统 - 前端

## 环境配置

1. 安装依赖：
```bash
npm install
```

2. 配置环境变量（可选）：
创建 `.env` 文件，设置：
```
VITE_API_BASE=http://localhost:8000
```

## 运行

启动开发服务器：
```bash
npm run dev
```

前端将在 http://localhost:5173 运行

## 构建

构建生产版本：
```bash
npm run build
```

## 功能

- **世界可视化**：显示虚拟城市和三个智能体（数学家、艺术家、工程师）
- **实时对话**：与智能体系统进行对话
- **状态同步**：实时同步智能体的位置、情绪、任务状态
- **角色选择**：可以选择特定智能体或自动分配
